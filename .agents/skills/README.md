# Skills DevMaster

Este diretório contém habilidades especializadas para trabalhar na solução DevMaster. Use estas habilidades para aumentar sua produtividade e garantir a consistência com os padrões estabelecidos no projeto.

## Habilidades Disponíveis

- **[Fundamentos do Framework](file:///c:/Users/jaime/source/repos/DevMaster/.agents/skills/framework_fundamentals/SKILL.md)**: Padrões arquiteturais principais, pontos de entrada e estado global da aplicação.
- **[DevExpress UI](file:///c:/Users/jaime/source/repos/DevMaster/.agents/skills/devexpress_ui/SKILL.md)**: Melhores práticas para controles DevExpress e layouts WinForms.
- **[Acesso a Dados](file:///c:/Users/jaime/source/repos/DevMaster/.agents/skills/data_access/SKILL.md)**: Execução de SQL, interações com banco de dados e gerenciamento de DataTable.
- **[Metadados e Estrutura (dmz)](file:///c:/Users/jaime/source/repos/DevMaster/.agents/skills/system_metadata/SKILL.md)**: Uso de tabelas de sistema (prefixo dmz) para descobrir a estrutura do banco de dados (tabelas, campos, relações e tipos).
- **[Padrões Comuns (Clientes)](file:///c:/Users/jaime/source/repos/DevMaster/.agents/skills/common_patterns/SKILL.md)**: Padrões de implementação frequentes encontrados nos projetos em src\Clientes, incluindo SelectSQL, loops e UI.
- **[Domínio ERP](file:///c:/Users/jaime/source/repos/DevMaster/.agents/skills/erp_domain/SKILL.md)**: Padrões de lógica de negócio para módulos específicos do ERP (Vendas, Financeiro, etc.).

## Como usar

Ao realizar tarefas no código DevMaster, consulte estas habilidades para:
1. Entender os padrões de codificação estabelecidos.
2. Usar utilitários existentes do framework em vez de reinventar a roda.
3. Seguir o padrão de "Ponto de Entrada" para customizações.
